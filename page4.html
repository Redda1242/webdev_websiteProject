Tasks

SWIPING THE CARD

This is a task which requires the player to swipe their card which they gat from their walled in a scanner. The player needs to make sure to swap the card at the right speed or the swipe would feel and they'll have to repeat again.

STABILIZE STEERING

This is task found in navigation room. To finish this task, the player needs to center a pointer randomly deviated from the center. By placing this pointer into the center. Then the task finishes. This task is a very simple one.

CHART COURSE

This is a short task located in Navigation, which requires you to put your finger on a mini spaceship at the left end of the screen, then go in zigzags. When it is at the other end of the screen, the task is complete.

EMPTY GARBAGE

This task has two parts. The player needs to dispose the spaceship trash from either the cafeteria or the oxygen room. Then completely get rid of it from the storage room. This is a visual task and crewmate can be confirmed innocent when doing this task.

UPLOAD DATA

To upload data, the player needs to firstly download data from a location depending on their task prompt then upload it in a different location. This task is a longer task.

DIVERT POWER

Divert power requires the player to divert power, this is in electricity, and then requires the player to accept the diverted power, which is in a location that depends on the task prompt.
